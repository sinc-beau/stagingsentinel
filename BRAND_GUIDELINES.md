# Sentinel Nexus Brand Guidelines

## Overview

Sentinel Nexus is an exclusive community connecting enterprise security leaders with solution providers through curated events, digital content, and strategic partnerships. Our brand embodies security, trust, innovation, and exclusivity.

---

## Brand Identity

### Mission
To create meaningful connections between CISOs, security executives, and trusted solution providers through high-value, solution-oriented engagements.

### Vision
To be the premier platform where security leaders collaborate, share insights, and discover innovative solutions to enterprise security challenges.

### Values
- **Trust & Security**: Building relationships based on verified credentials and mutual respect
- **Quality Over Quantity**: Curated experiences with vetted participants
- **Solution-Oriented**: Focus on addressing real challenges, not product pitches
- **Innovation**: Embracing cutting-edge security technologies and strategies
- **Community**: Fostering genuine connections among security leaders

## Visual Identity

### Color Palette

#### Primary Colors
- **Teal (#14b8a6)**: Primary brand color - represents trust, innovation, and security
  - Usage: Primary CTAs, accents, interactive elements
- **Cyan (#22d3ee)**: Secondary accent - represents technology and digital transformation
  - Usage: Gradients, highlights, secondary accents
- **Black (#000000)**: Base background - represents sophistication and focus
  - Usage: Primary backgrounds, overlays

#### Neutral Colors
- **Slate 950 (#020617)**: Deep background
- **Slate 900 (#0f172a)**: Card backgrounds
- **Slate 800 (#1e293b)**: Input fields, elevated surfaces
- **Slate 700 (#334155)**: Borders
- **Slate 400 (#94a3b8)**: Secondary text
- **Slate 300 (#cbd5e1)**: Body text
- **White (#ffffff)**: Headlines, primary text

#### Accent Colors (Use Sparingly)
- **Yellow (#eab308)**: Diamond/Premium tier
- **Purple (#a855f7)**: A La Carte tier
- **Blue (#3b82f6)**: Gold/Connector tier
- **Red (#ef4444)**: Errors, urgent notifications

### Typography

#### Font Families
- **Primary**: System font stack (-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif)
- **Monospace**: Used for technical labels and tags (font-mono)

#### Hierarchy
- **Hero Headlines**: 6xl-8xl (text-6xl to text-8xl), font-black (900), tight leading
- **Section Titles**: 4xl-5xl (text-4xl to text-5xl), font-black (900)
- **Subsection Titles**: 2xl-3xl (text-2xl to text-3xl), font-bold (700)
- **Body Text**: base-xl (text-base to text-xl), font-normal (400)
- **Labels**: xs-sm (text-xs to text-sm), font-semibold (600), uppercase, tracking-wider

### Logo & Branding

#### Primary Logo
- **"SENTINEL NEXUS"** wordmark
- Always displayed in uppercase
- Primary version: White text
- Gradient version: Teal to Cyan gradient on text
- Shield icon accompanies the wordmark in key placements

#### Logo Usage
- Minimum clear space: Equal to the height of the letter "S"
- Never distort, rotate, or alter the logo proportions
- Maintain high contrast with background
- Use gradient version for hero sections and key branding moments

### Iconography

#### Icon Style
- Use Lucide React icon library
- Line style: 2px stroke weight
- Size: Typically 16px-24px (w-4 h-4 to w-6 h-6)
- Color: Match to context (teal for primary actions, slate for secondary)

#### Key Icons
- **Shield**: Security, protection, trust
- **Lock**: Exclusivity, privacy
- **Users/Target**: Community, audience
- **Zap/Lightning**: Innovation, speed
- **Award/Star**: Premium, excellence
- **Network**: Connections, community

## Content Guidelines

### Voice & Tone

#### Voice Characteristics
- **Professional**: Industry expertise, credible
- **Direct**: Clear, concise messaging
- **Exclusive**: Premium positioning
- **Trustworthy**: Security-focused, reliable

#### Tone Variations
- **Hero Sections**: Bold, confident, aspirational
- **Feature Descriptions**: Informative, benefit-focused
- **CTAs**: Action-oriented, clear value
- **Forms**: Helpful, reassuring

### Writing Style

#### Headlines
- Use active voice
- Focus on benefits, not features
- Keep it bold and impactful
- Examples:
  - "PARTNER WITH SENTINEL NEXUS"
  - "CONNECT WITH SECURITY LEADERS"
  - "SOLUTION-ORIENTED ENGAGEMENT"

#### Body Copy
- Short paragraphs (2-3 sentences max)
- Bullet points for scannability
- Emphasize outcomes and value
- Avoid jargon unless audience-appropriate

#### Calls to Action
- Clear and specific
- Action verbs (Join, Connect, Request, Explore)
- Examples:
  - "REQUEST PARTNERSHIP INFO"
  - "JOIN THE GUILD"
  - "EXPLORE OPPORTUNITIES"

## Interactive Elements

### Animations

#### Entrance Animations
- Fade in with slight upward movement (y: 20 to 0)
- Stagger animations for lists (delay: index * 0.1s)
- Duration: 0.6s with ease-out

#### Hover Effects
- Scale: 1.02-1.05 for cards and buttons
- Border color transitions
- Shadow enhancements
- Duration: 0.3s

#### Background Effects
- Subtle grid patterns (opacity: 5%)
- Scanning line animations (10s linear infinite)
- Gradient glows (blur-3xl, opacity: 3%)
- Network node pulses (3s duration)

### Responsive Behavior

#### Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

#### Mobile Optimizations
- Simplified animations
- Hamburger navigation
- Stacked layouts
- Larger touch targets (minimum 44px)

## Accessibility

### Standards
- WCAG 2.1 AA compliance
- Minimum contrast ratio 4.5:1 for body text
- Minimum contrast ratio 3:1 for large text
- Keyboard navigation support
- Screen reader compatible

### Best Practices
- Alt text for all images
- ARIA labels for interactive elements
- Focus indicators visible
- Form labels clearly associated
- Semantic HTML structure
